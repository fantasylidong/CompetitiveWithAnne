# ✅ HexTags Lite v1.0.1 - 更新完成

## 🆕 新增功能

### 自动隐藏管理员命令

现在所有以 `!` 或 `/` 开头的命令都不会显示在聊天中！

**隐藏的命令示例：**
```
!rygive @me weapon_rifle_ak47
/kick player
!sm_ban
!reloadtags
!ch
```

**普通聊天正常显示：**
```
你好大家
GG
这关太难了
```

## 📝 工作原理

插件会在处理聊天消息前检查第一个字符：
- 如果是 `!` 或 `/` → 不显示（让SourceMod处理命令）
- 其他内容 → 正常显示带称号的聊天

## 🎯 优势

- ✅ **更干净的聊天**：不再有管理员命令刷屏
- ✅ **保护隐私**：玩家看不到管理员在执行什么操作
- ✅ **提升体验**：聊天只显示真实交流内容
- ✅ **自动处理**：无需配置，开箱即用
