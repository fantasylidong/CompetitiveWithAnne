# HexTags Lite v1.0.2 - 智能命令过滤

## 🆕 新功能

### 智能命令显示系统

插件现在使用**白名单机制**来控制哪些命令可以显示：

#### ✅ 允许显示的命令（白名单）
```
!vote / /vote           - 投票命令
!rtv / /rtv             - 换图投票
!nominate / /nominate   - 提名地图
!revote / /revote       - 重新投票
```

#### ❌ 自动隐藏的命令
```
!rygive                 - 管理员给予物品
!sm_ban                 - 管理员封禁
!sm_kick                - 管理员踢出
!reloadtags             - 重载配置
!ch / !chenghao         - 称号菜单
!setch                  - 自定义称号
... 所有其他管理员和插件命令
```

#### ✅ 正常显示的聊天
```
你好大家
GG
这关太难了
2333
... 所有普通文字消息
```

## 🎯 工作原理

1. **检测格式**：判断消息是否以 `!` 或 `/` 开头
2. **白名单匹配**：检查是否在允许列表中
3. **权限判断**：管理员命令自动隐藏
4. **显示决策**：
   - 白名单命令 → ✅ 显示
   - 管理员命令 → ❌ 隐藏
   - 其他插件命令 → ❌ 隐藏
   - 普通聊天 → ✅ 显示

## 🔧 自定义白名单

如需添加更多可显示的命令，编辑源码中的 `ShouldHideCommand` 函数：

```sourcepawn
static const char allowedCommands[][] = {
    "!vote",
    "/vote",
    "!rtv",
    "/rtv",
    "!nominate",
    "/nominate",
    "!revote",
    "/revote",
    // 添加你的命令：
    // "!你的命令",
    // "/你的命令"
};
```

## 📊 效果对比

### 玩家视角
| 输入 | 显示效果 |
|------|----------|
| `!vote` | ✅ 显示：`<称号> 玩家名 : !vote` |
| `!rtv` | ✅ 显示：`<称号> 玩家名 : !rtv` |
| `!rygive @me rifle` | ❌ 不显示（命令静默执行） |
| `!ch` | ❌ 不显示（打开菜单） |
| `你好` | ✅ 显示：`<称号> 玩家名 : 你好` |

### 管理员视角
- **白名单命令**：和普通玩家一样显示
- **管理员命令**：自动隐藏，保护操作隐私
